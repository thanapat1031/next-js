# NEXT-JS

### Main Dependencies
 - next (4.0.5)
 - react (16.0.0)
 - react-dom (16.0.0)

### Tools
 - npm (5.4.2)
 - yarn (1.2.1)
 - node (8.7.0)

## Basic commands

#### Clone repository

`git clone https://github.com/thanapat1031/next-js.git`

or

`git clone git@github.com:thanapat1031/next-js.git`

#### Install dependency

`yarn install`

#### Start development server

`yarn dev`

#### Build production

`yarn run build`
